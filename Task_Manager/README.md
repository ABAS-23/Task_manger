# Task Manager

## Description
A simple Task Manager built with Python to practice adding, viewing, and marking tasks as done, along with basic data handling and user interaction.

## Features
- Add new tasks
- View all tasks
- Mark tasks as done
- Simple console-based interface
- Uses list of dictionaries for task management

## How to Run
1. Make sure Python is installed on your computer.
2. Run the main Python file:
   ```bash
   python task_manager.py
